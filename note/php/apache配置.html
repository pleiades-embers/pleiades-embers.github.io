    # apache 配置

    #装载php模块 其实就是php语言包中的一个文件 需要给出php7apache2_2.dll的绝对路径 LoadModule php7apache2_4 "D:/xampp/php/php7apache2_4.dll"

    #告知apache, 凡是 “.php”的后缀文件,找到php语言模块 AddType application/x-httpd-php .php .pad .phtml

    # 配置php的时区

    #图文详解xampp集成环境中PHP时区的设置

    在htdocs文件夹创建phpinfo.php文件，查看php配置信息，文件代码如下

    `<?php phpinfo();?>`
    1

    启动Apache服务器，在浏览器中输入如下地址 http://localhost/phpinfo.php 在此页面上查找timezone,如图

    可以看到，默认时区是Europe/Berlin 打开php.ini文件，搜索Berlin,改成 date.timezone=PRC,并保存

    重新启动Apache服务器，刷新页面，如图

    可以正常显示中国时区的时间了

    # 设置默认站点路径

    搜索 DocumentRoot 在Apache httpd.conf 中修改

    <Directory "要设置权限的目录路径">
	Options Indexes #设置当一个请求中没有给定请求的文件名又没有默认网页（首页）的时候,显示文件列表
	Order Deny,Allow #设置权限控制的先后顺序,只有2种: Deny,Allow(先拒绝,后允许) ,Allow,Deny(先允许,后拒绝)
	#用于设定要拒绝访问的来源地址或服务器名,几乎没有实际意义
	Deny from 拒绝名单 比如:deny from 192.168.3.8 172.180.4
	Allow from 允许名单 比如:Allow from all
	#设置该文件夹下的 "默认网页“ （首页）,可以设置多个用于请求中不带文件名的时候自动使用该文件作为”返回" 页面
	DirectoryIndex index.php index.html main.php default.php abc.php 123.html
	# 下一行,表示当前目录 ,允许 ”分布式” 权限配置
	AllowOverride all
</Directory>
